# 📚 Lei de De Morgan e Teoria dos Conjuntos

Este repositório tem como objetivo apresentar, de forma clara e didática, os conceitos fundamentais da **Teoria dos Conjuntos** e as **Leis de De Morgan**, amplamente utilizadas na matemática discreta, lógica e computação.

---

## 📌 1. Teoria dos Conjuntos

A Teoria dos Conjuntos é um ramo da matemática que estuda coleções de elementos chamadas **conjuntos**.

### 🔹 Definição
Um conjunto é uma coleção bem definida de objetos.

**Exemplo:**
- A = {1, 2, 3}
- B = {2, 3, 4}

---

### 🔹 Operações com Conjuntos

#### ➤ União ( ∪ )
Conjunto formado por todos os elementos de A ou B.

\[
A ∪ B = \{x | x ∈ A \text{ ou } x ∈ B\}
\]

**Exemplo:**
A ∪ B = {1, 2, 3, 4}

---

#### ➤ Interseção ( ∩ )
Conjunto formado pelos elementos comuns entre A e B.

\[
A ∩ B = \{x | x ∈ A \text{ e } x ∈ B\}
\]

**Exemplo:**
A ∩ B = {2, 3}

---

#### ➤ Diferença ( - )
Elementos que estão em A e não estão em B.

\[
A - B = \{x | x ∈ A \text{ e } x ∉ B\}
\]

---

#### ➤ Complemento ( Aᶜ )
Elementos que não pertencem ao conjunto A dentro de um universo U.

---

## 📌 2. Leis de De Morgan

As Leis de De Morgan são regras fundamentais que relacionam **negação, união e interseção**.

---

### 🔹 Primeira Lei de De Morgan

\[
(A ∪ B)^c = A^c ∩ B^c
\]

👉 O complemento da união é igual à interseção dos complementos.

---

### 🔹 Segunda Lei de De Morgan

\[
(A ∩ B)^c = A^c ∪ B^c
\]

👉 O complemento da interseção é igual à união dos complementos.

---

## 📌 3. Interpretação Intuitiva

- "Não (A ou B)" = "Não A e Não B"
- "Não (A e B)" = "Não A ou Não B"
